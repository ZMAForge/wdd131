Overview

Purpose

My purpose in creating this page is to have an easily updatable collection of all the projects that I have done, essentially creating a digital portfolio. As I learn more about how forms submit and how to utilize that, I intend to be able to add to the collection of projects through the form submission instead of manually adding new object.

Audience

My intended audience are myself and anyone who has an interest in my work and what I should do in the future. Simply put my audience is myself, my family, and future employeers.

Dynamic elements

The Dynamic Elements I will be using will be for the main content of the page. I plan for the main body to be a collecion of cards, that will be populated as a means of display at first, but then by changed based off the users selection.
